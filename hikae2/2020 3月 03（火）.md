rbenv でバージョンを切り替える方法は２つ。GlobalかLocalか。

Globalで切り替わらないのにLocalで替わる、というのが今回の問題。versionsファイルというのが原因になっていることがある、とか。

local でだけ替わるなら、フォルダを変えたときはまた常に変え直すこと。

```Plain
rbenv global 2.6.5
    パソコン全体で使うRubyバージョンを一括で切り替え。これが不発。


rbenv versions
    rbenv で管理しているRubyのバージョンすべてを一覧する。今回はこれで6.3 と6.5がともに表示されていたが、6.5に変えるコマンドを打ってもエラーが無いのに変更されない。


rbenv install 2.6.5
    これでrbenv を経由してRubyをインストールする。だが、今回はすでにPCには6.5がインストール済みだったため不発。というかY/Nダイアログが続く。


rbenv init
    bashファイルがおかしいときはこれらを使う。今回はインストール先フォルダ自体は適切だったので不発。


rbenv local 2.6.5
    これなら通った。プロジェクトフォルダごとに切り替えるコマンド。


which ruby
    /Users/apple/.rbenv/shims/ruby　インストール先はこれでヨシ！


```
